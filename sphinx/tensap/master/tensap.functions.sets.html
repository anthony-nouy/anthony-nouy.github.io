Module is_in.

tensap.functions.sets.is_in.is_in(B, x)¶

Return a boolean array of length x.shape[0] such that ok[k] = True if the point x[k, :] is in the box given by B.

B is of shape (2, dim) and contains the two extreme points B[0, :] and B[1, :] of the box.

Parameters
Bnumpy.ndarray

The box.

xnumpy.ndarray

The points.

Returns
oknumpy.ndarray

A boolean array indicating if the points x are in the box B.
